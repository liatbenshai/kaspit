# כספית - קבצים מתוקנים

## סיכום השינויים

### קובץ: `app/(dashboard)/income/page.tsx`

#### תיקון 1: קישור חשבוניות חכם
- **לפני:** המערכת הציגה את *כל* חשבוניות המס ללא קישור, בלי שום סינון
- **אחרי:** הוספתי פונקציות `getRelevantTaxDocuments` ו-`getRelevantBusinessInvoices` שמחשבות ציון התאמה:
  - +50 נקודות: אותו לקוח
  - +40 נקודות: סכום זהה (±1%)
  - +30 נקודות: סכום קרוב (±5%)
  - +10 נקודות: תאריך קרוב (עד 7 ימים)
  - המסמכים מוצגים ממוינים לפי ציון התאמה
  - אפשרות להציג רק מסמכים עם התאמה טובה (≥30 נקודות) או הכל

#### תיקון 2: הסרת קישור
- **לפני:** אחרי קישור, אי אפשר היה לתקן טעויות
- **אחרי:** 
  - כפתור הסרת קישור (Link2Off) בטבלה ליד כל מסמך מקושר
  - פונקציית `handleUnlinkDocument` שמסירה קישור ופותחת מחדש את חשבונית העסקה
  - באופן עריכה - בחירת "הסר קישור" ב-Select
  - אזהרה למשתמש שחשבונית העסקה תחזור לסטטוס פתוח

#### תיקון 3: שיפור ממשק קישור
- מודל קישור חדש עם:
  - הצגת פרטי המסמך לקישור (מספר, לקוח, סכום)
  - תצוגה ויזואלית של ציון התאמה (מצוינת/טובה/חלשה)
  - הצגת הפרש סכום באחוזים
  - סימון "✓ אותו לקוח" או "✗ לקוח שונה"
  - מספר ימי הפרש בין התאריכים
  - צביעה לפי רמת התאמה (ירוק/כחול/כתום)

#### תיקון 4: טיפול בעריכה
- כשעורכים מסמך עם קישור:
  - אם מסירים קישור - חשבונית העסקה נפתחת מחדש
  - אם משנים קישור - הישנה נפתחת, החדשה נסגרת
  - אם מוסיפים קישור חדש - חשבונית העסקה נסגרת

#### תיקון 5: טיפול במחיקה
- כשמוחקים מסמך מקושר - חשבונית העסקה המקושרת נפתחת מחדש
