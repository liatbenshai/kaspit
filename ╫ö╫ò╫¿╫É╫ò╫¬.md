# תיקון זמני בית/משפחה - גרסה סופית

## 📋 מה תוקן?

### 1. ✅ Google Calendar
בחירת היומן נשמרת גם אחרי רענון.

### 2. ✅ זיהוי משימות בית/משפחה
עכשיו המערכת מזהה נכון משימות בית לפי ה-task_type:
- `shaked`, `dor`, `ava`, `yaron` → משפחה
- `cleaning`, `cooking`, `laundry`, `shopping` → בית

### 3. ✅ תזמון משימות בית ביום שישי
משימות בית מתוזמנות עכשיו בזמנים הנכונים:
- **ימים א'-ה':** 17:00-21:00
- **שישי-שבת:** 08:00-22:00 (כל היום)

### 4. ✅ חישוב שעת התחלה אוטומטית
כשמוסיפים משימת בית, השעה מחושבת לפי זמני בית ולא זמני עבודה.
